## 第一讲：生物技术药物概论

### 生物技术药物简介

生物技术药物：采用**现代生物技术**研制的蛋白类或核酸类药物（主要包括重组蛋白药物，单克隆抗体药物，新型疫苗）

生物制药技术：用于制药的现代生物技术，主要包括基因工程、酶工程、抗体工程、细胞工程、发酵工程等

传统制药：
- 抗生素药物
- 氨基酸药物
- 维生素药物

现代生物技术制药（生物制药）
- 重组蛋白药物
- 单克隆抗体药物
- 疫苗
- 核酸药物

生物药物的特点:
- 优点
    + 治疗针对性强
    + 药理活性高
    + 毒副作用小
- 缺点
    + 结构复杂、分离纯化难、成本高
    + 结构不稳定、制剂复杂
    + 生理副作用常有发生（免疫反应和过敏反应）

行业特点：高技术、高投入、长周期、高风险、高收益

### 生物技术药物的发展历程

生物技术药物的基础——基因工程

理论上的三大发现：
- 格里菲斯——遗传物质是DNA
- DNA双螺旋结构和半保留复制机制
- 三联密码子和中心法则

技术上的三大发明：
- 工具酶：限制性核酸内切酶、DNA连接酶、
- 载体：质粒、F因子、R因子、CoE
- 逆转录酶 提供目的基因

1973年基因工程元年：斯坦福Cohen成功重组转化

### 基因工程制药的过程和内容

上游工程：
- 获得目的基因
- 组建重组质粒
- 构建工程菌（或者细胞）

下游技术：
- 培养工程菌
- 产物分离纯化
- 除菌过滤
- 半成品检定
- 成品检定
- 包装

基本过程:
1. 分离目的基因
2. 切割目的基因和载体
3. 连接目的基因和载体
4. 转化到宿主细胞
5. 筛选出高表达阳性克隆体
6. 表达出产物

基因工程四要素：
- 基因
- 工具酶
- 载体
- 表达体系

目的基因的获取：
- 化学合成
    + 已知序列
    + 固相合成
- 基因文库
    + gene pool: 种群的全部基因
    + gene bank
    + gene library: 把一个种群的基因转化到微生物中，保存在实验室里，有载体
    + 基因文库的筛选：
        + 用标记的DNA探针进行DNA杂交
        + 抗敌对蛋白质进行免疫杂交
        + 对蛋白质的活性进行鉴定
    + 真核生物目的基因的获得
        + cDNA的方法
        + PCR方法
- 反转录法(cDNA文库)
    1. 分离表达目的基因的组织或细胞
    2. 从组织或细胞中制备总体RNA和mRNA
    3. 合成第一条cDNA链，需要RNA模板......<++>
    4. <++>
- 反转录PCR

基因工程工具酶：
- 主要是2型限制性核酸内切酶
- 识别特异性性序列，并按照一定规则进行切割
- 产生粘性末端或者平末端

老师上课突然的idea:
- 依靠DNA合成的原理，对碱基进行改造，然后提供DNA模板，供其合成；不同模板可以产生不同高分子聚合物

载体：
- 能在宿主细胞中进行自我复制和表达
- 表达载体、克隆载体
- 原核载体、真核载体

克隆载体：
- 复制起始区
- 抗性基因
- 酶切位点

表达载体：
- 启动子
- 乳糖操纵子
- 克隆载体的一些东西

理想克隆载体的条件:
- 自我复制能力
- 适宜的限制酶切位点
- 一个或多个标记基因
    + 抗氨苄青霉素基因
    + 抗四环素基因
- <++>

质粒：
- PET238
- 柯斯质粒
- YAC质粒：酵母人工染色体
- 穿梭载体：可以在真核和原核中表达

表达体系：
- 原核表达
- 真核表达
    + 酵母
    + 动植物细胞
- 其它表达体系

## 抗体工程制药

1. 抗体发展三个阶段：
- 血清
- <++>
- 单抗

2. 单抗研发的演变：
- 100%鼠源单抗
- 鼠人嵌合抗体
- 人源化抗体（~0.95）
- 100%人源抗体

3. 后天免疫反应的四个阶段
- 遭遇
- 动员
- 扫荡（抗体）
- 休止

4. 抗原：能引起抗体生成的物质；抗体：能与抗原特异结合的具有免疫活性的功能性球蛋白

5. 抗原决定基：一个抗原分子可能有多个抗原决定基，每个抗原决定基可诱发至少一种专一性抗体，蛋白质性质的抗原决定基含有6个以上氨基酸

6. 多克隆抗体：由多个淋巴B细胞产生的多种抗体的混合物，针对多种抗原决定簇的抗体

7. 单克隆抗体：由单个B淋巴细胞产生的结构和特异性完全相同的高纯度抗体，针对单一抗原决定簇

## 单克隆抗体生产技术

1. 杂交瘤技术：
- B细胞与骨髓瘤细胞融合，产生杂交瘤细胞，既可无限繁殖，又可产生抗体
- 筛选得到阳性细胞之后，打入小鼠腹腔表达抗体或者通过细胞培养表达抗体
- PEG诱导融合法
- HAT培养基筛选

2. 阳性克隆的筛选：
- 免疫酶法
- 免疫荧光技术
- <++>

3. 杂交瘤细胞和抗体性质鉴定
- 染色体分析
- 抗体性质鉴定和分析

4. 纯化：
- 离心
- 凝胶过滤
- 阴离子交换层析
- 亲和层析

5. 单抗的应用： 
- 用于科学研究
- 医疗诊断
- 免疫检验

## 基因工程抗体

- B细胞熟化：DNA重排，以产生丰富的抗体基因

基因工程改造抗体：
- 小分子抗体
    + Fab，Fv，ScFv，单域抗体，最小识别单位
    + 细菌发酵生产
    + 分子小，穿透力好
    + 循环时间短
    + 容易改造
- 抗体融合蛋白
- 单抗

抗体库技术：克隆出全部抗体的轻、重链可变区基因克隆出来，在原核载体上表达，然后筛选出所需的特异基因和抗体
- RT-PCR：克隆全套抗体可变区基因
- 抗体基因片段在宿主菌中的功能性表达
- 噬菌体展示技术
- 基本上是人源抗体
- 检测，加治疗
